WhoAmI — Random Profile Generator
So I needed a break from building a full stack fitness app and decided to build something fun and low stakes. WhoAmI is a random social media profile generator — click a button and get a completely fake profile with a name, username, bio, follower count, interests, and a photo. Every click gives you something new.
No API. No backend. No database. Just React, some hardcoded data arrays, and Math.random() doing all the work.
Built this as a side project to practice component architecture and state management without all the heavy lifting of a full stack build. Turned out pretty clean though.
Built with:

React
React Router
CSS

Features:

Generate a random profile with one click
Save profiles to a wall page
Clean light UI inspired by modern social platforms
